General

Global options

Determines the contrast percentage of text in Dark mode or Reverse colors.
46% equals normal contrast in Dark mode.
50 is 100% contrast in Reverse Colors.

Determines the brightness of text and some background in Dark mode or Reverse colors.
0% equals normal brightness.

Adjusts the contrast of text color.
If the text becomes white against dark backgrounds, try lowering this value. Reverse for black on bright background.
Also sets the Brightness % in Dark mode, with 0 being normal brightness.
If strength is more than 200, most colors are made black on light background or white on dark background.
If strength is ODD number, INPUT text boxes color is changed to maximum contrast white/black.
In Dark mode or Reverse colors, this value (within -200 to 200 for reverse colors and -300 to 300 for Dark mode) determines "brightness" of text / backgrounds.

Increases font size. Fonts above the size threshold won't be affected.
Preset value is at 14%

Determines the maximum size a font has to be (in pixels) for it to change size.
By default, set at 18px.
If high value, 25/30/40, if font increase % > 0, text will be HUGE.

Changes the font weight of all text to specifed slider value (400 is normal). 100 is thinnest and 900 is boldest text.
