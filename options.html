General

Activate on all websites

Toggles the add-on automatically on every website possible.
When this is disabled, the plugin will be enabled only on whitelisted websites.

Global options

For Dark mode

Increases text contrast in a dark theme or dark background.
e.g. Light grey turns whiter. Colorful text can be made more / less colorful.
The effect can be adjusted with the Contrast and Brightness sliders.

Contrast (%)

Determines the contrast percentage of text in Dark mode or Reverse colors.
46% equals normal contrast in Dark mode.
50 is 100% contrast in Reverse Colors.

Brightness (%)

Determines the brightness of text and some background in Dark mode or Reverse colors.
0% equals normal brightness.

Strength

Adjusts the contrast of text color.
If the text becomes white against dark backgrounds, try lowering this value. Reverse for black on bright background.
Also sets the Brightness % in Dark mode, with 0 being normal brightness.
If strength is more than 200, most colors are made black on light background or white on dark background.
If strength is ODD number, INPUT text boxes color is changed to maximum contrast white/black.
In Dark mode or Reverse colors, this value (within -200 to 200 for reverse colors and -300 to 300 for Dark mode) determines "brightness" of text / backgrounds.

Increase font (%)

Increases font size. Fonts above the size threshold won't be affected.
Preset value is at 14%

Size threshold (px)

Determines the maximum size a font has to be (in pixels) for it to change size.
By default, set at 18px.
If high value, 25/30/40, if font increase % > 0, text will be HUGE.

Font Weight

Changes the font weight of all text to specifed slider value (400 is normal). 100 is thinnest and 900 is boldest text.

Avoid colorful text

Attempts to avoid colorful text and links, if they are having low contrast and brightness.
Can be adjusted with "Strength".

Skip headings

Skips applying setting to HTML headings. (<h1>, <h2>, <h3>)

Skip NAV & SECTION

Skips widening of SECTION and increasing height of NAV elements.
NAV elements are usually in menus and SECTION in a big part of webpage
If "Force Reverse Colors" is enabled, images within NAV and HEADER parts of page are skipped.
( This only applies if "Invert Image colors" is enabled ).

Skip text height changes

Skips increasing height of text whose height is less than font size, and unset line height of changed text.
Enable this if there is height between text lines is increased.

Skip white text

Skips applying settings on white text.
White text is usually behind images, where determining the background color is not always possible.

Skip nodes with stylesheets rules

Applies font and color changes of stylesheets of the webpage
Nodes which are done by stylesheet are skipped in processing.
Enable to increase processing speed.

ALL CAPITAL TEXT

Makes all text in CAPITALS.

Starting 3 letters bold

Makes start 3 letters bold of all words on page.

Skip links in 3 letters bold

Avoid changing webpage links when start 3 letters bold is enabled.

Underline black links / Change Font

Adds an underline to links that are black, to distinguish them from other black text.
Also changes ALL fonts to Sans-serif style, or to Serif style if Input Border is enabled.
If Input Border is also enabled in Dark mode, the brightness / contrast settings apply to more elements and images (but colors of some images get distorted).

Force Reverse Color

Inverts the colors of a page, causing light webpages to become dark and dark pages to become light.
Can be used for making light background sites Dark background.

Invert Image Color

Inverts the colors of images, which when Reverse Colors is set, makes images appear normal.

Reverse colors hue correction

Changes colors of text and some backgrounds to match original website in reverse colors.

Force max opacity

Some websites use dark fonts that still look grey because of low opacity. This option forces full opacity on all text.

Disable this on websites that use opacity to hide some elements (e.g. YouTube).
Also forces background color of page to reverse color when "Force Reverse Colors" is enabled and the background color is not "reversed".

Colorblind adjust colors

Change text colors to make them readable for colorblind condition.
Adjusts red and green values of color to exchange with blue value.

Input border

Attempts to add a black outline around text areas.
Also changes All Fonts Style to Serif if Underline links is enabled.
If Input Border is also enabled in Dark mode, the brightness / contrast settings apply to more elements and images (but colors of some images get distorted)
