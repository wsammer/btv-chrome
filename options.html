General

Global options

Determines the contrast percentage of text in Dark mode or Reverse colors.
100% contrast is normal / default.

Determines the brightness of text and some background in Dark mode or Reverse colors.
100% equals normal / default brightness.

1. Adjusts the contrast of text color.
2. Sets the zoom % of "Shift + zoom".
3. If ODD value, sets the high contrast color of input textboxes.

If the text contrast is very high or to get original colors, decrease this to high negative values.
For getting high(est) contrast, increase this to high positive value 100 to 300.
If strength is ODD number, input textboxes color is changed to maximum contrast white/black.

Increases font size. Fonts above the size threshold won't be affected.
Increase in width of "SECTION" and "NAV" elements depends on font increase % unless "Skip NAV & SECTION" is enabled.
Preset value is at 14%

Determines the maximum size a font has to be (in pixels) for it to change size.
By default, set at 18px.
If high value, 25/30/40, if font increase % > 0, text will be HUGE.

Changes the font weight of all text to specifed slider value (400 is normal). 100 is thinnest and 900 is boldest text.
Changes the "inverse text size" calculation to "normal text increase" when "Start 3" disabled and "Skip links" enabled.
400 is default "inverse text size increase" and decreases as weight value is decreased to 300 .. 100 .. 900 .. 500, with 500 being "normal text size increase" which is closest to normal website.
